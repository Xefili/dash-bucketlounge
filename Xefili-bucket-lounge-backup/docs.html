Bucket Helper Command Syntax

Basics: The default bot prefix is “%”
Some commands require one or more arguments!

Commands without the prefix:
!All of the following commands can be run by everyone!
All of the following commands do not accept any arguments!

‘vc’: Pings @Staff and deletes original message.
‘is bot off’: What do you expect it to do?
‘is bot on’: Shows the current status of the Bot; Status.

Commands with the prefix:
!Some of these commands require Administrator permissions and require one or more arguments!

‘%ping’: Replies with Pong!
‘%hello’: Hello!
‘%help’: Shows help embed.
‘%randomnumbergenerator’: Replies with a random number in the specified range. Alias: %rng . Usage %rng a b
‘%staff’: Shows an embed with all Bucket Lounge staff members.
‘%youtube’: Shows an embed with the Bucket Lounge Youtube Channels. Alias: %yt . %kick’: Kicks a member for a specified reason. Usage: %kick . This command requires you to have the Kick Members permission!
‘%ban’: Bans a member for a specified reason. Usage: %ban . This command requires you to have the Ban Members permission!
‘%link_embedder’: Embed the specified link in an embed. Usage: %link_embedder < link>. Multiple words in must be in quotation marks like this: “multi word statement”
‘%repeat’: Repeats the specified text. Usage: %repeat .
